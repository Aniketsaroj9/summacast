## Threat Model Analysis (ASVS Level 1)

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Cross-Site Scripting (XSS) via transcripts or QA response | High | Rely on React's default text rendering (`{variable}`) to auto-escape text. Never use `dangerouslySetInnerHTML` for raw transcripts or QA answers unless verified safe. |
